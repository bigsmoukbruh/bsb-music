**BSB Music — десктопный аудиоплеер на .NET Windows Forms для потокового воспроизведения музыки из удаленного JSON-плейлиста с кэшированием и консольным управлением.**
**Функционал**
**1. Загрузка плейлиста из JSON**
Приложение загружает список треков с удаленного сервера в формате JSON. Поддерживается структура с названием трека, авторами (включая флаг "иноагент") и прямой ссылкой на MP3-файл. Кнопка ручной загрузки и команда reload позволяют обновить плейлист без перезапуска программы.

**2. Кэширование MP3**
При первом прослушивании трек скачивается и сохраняется в локальную папку Cache. При повторном воспроизведении файл берется из кэша — трафик не расходуется. Поврежденные файлы автоматически перезагружаются. Команда clearcache полностью очищает папку с кэшем.

**3. Управление воспроизведением**
Play/Pause — кнопка запуска и паузы

* Prev / Next — переключение на предыдущий или следующий трек

* Автоматический переход к следующему треку по окончании текущего

**4. Перемотка**
* Клик по прогресс-бару — мгновенная перемотка на выбранную позицию

* Кнопки ±5 секунд — отмотка назад или вперед на 5 секунд

* Отображение текущего времени и общей длительности трека

**5. Поиск / фильтрация треков**
* Ввод текста в поле поиска и нажатие Enter фильтрует список треков по названию или имени исполнителя. Отображаются только подходящие композиции.

**6. Консольные команды (ввод в поле поиска + Enter)**
* Команда	Действие
* help	Показать окно со списком всех команд
* reload	Принудительно перезагрузить плейлист с сервера
* clearcache	Остановить плеер и удалить все файлы из папки Cache
* bsbconsole	Открыть/закрыть отдельное окно консоли отладки
* changedisk [имя]	Сменить образ диска (из папки Assets, например changedisk vinyl)
**7. Консоль отладки (bsbconsole)**
* Отдельное окно с логами в реальном времени:

* Зеленый текст на темном фоне

* Временные метки у каждого сообщения

* Логируются: загрузка треков, воспроизведение, ошибки, команды, работа кэша

* Кнопки "Очистить" и "Закрыть"

* Автоматическая прокрутка к новым сообщениям

**8. Вращение диска**
Во время воспроизведения изображение диска в интерфейсе вращается. Включение/отключение вращения можно настроить через файл settings.json (параметр EnableDiscRotation). Скорость вращения фиксированная.

**9. Отображение авторов с меткой "иноагент"**
При загрузке JSON поддерживается массив авторов. Если у автора стоит флаг foreign_agent: true, рядом с его именем в интерфейсе отображается отметка (иноагент). Список авторов выводится под названием трека в кнопках плейлиста.
